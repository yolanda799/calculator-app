# Calculator App Front-End

Front-end project of a calculator application, developed for study and practice purposes.  
Built using **HTML5**, **CSS3**, and **JavaScript**, focusing on layout, styling, and basic interactivity.  
This project has no advanced functionality and is for educational purposes only.

## Features
- Fully functional front-end calculator
- Responsive layout
- Styled with CSS3
- Basic interactivity with JavaScript
- Educational project for practice purposes
